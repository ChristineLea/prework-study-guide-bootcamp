HTML

  • HTML stands for HyperText Markup Language
  • It defines the meaning and structure of a web page.
  • HTML elements are used to "markup" the content for the web browser to display.
  • Every HTML document must begin with the !DOCTYPE html declaration.
  • This notifies the browser that the content of this file is HTML.
  • The html element wraps all the content on the page.
  • The content within the head element is not displayed by the web browser.
  • It contains content such as SEO keywords, page title, and links to CSS stylesheets.
  • The title element sets the title of the page that will appear in the browser tab.
  • It is nested within the head element.
  • The body element contains all the content that displays will be displayed on the page.
  • Semantic HTML elements are elements which describe the content they contain.
  • The main, section, header, and nav are all examples of semantic elements.
  • Heading elements range from h1 (the most important) to h6 (the least important).
  • The img, or image element, is used to add an image to an HTML document.
  • Links can be added to an HTML document using the a or anchor element.
  • Lists, such as this one, can be either unordered (bullet point) or ordered (numbered).
  • Both the ul element (unordered list) & the ol element (ordered list) are nested with li elements.
  • The li element stands for list item, and the content, such as this sentence, is wrapped between li tags.

CSS

  • CSS is used to add styles to a web page.
  • Margin refers to the amount of space outside of an element.
  • It can be used to push elements away from one another.
  • Padding refers to the amount of extra space around an element's content.
  • Border property shorthand will style the thickness of the border, border style, and border color.
  • Width and Height properties refer to the dimensions of an element.
  • Font-size sets the size of the text.
  • Background-color refers to the background color which fills an element/s.
  • Color refers to the color of the content text.
  • Box-shadow property can add a shadow to an element by defining values along X-axis & Y-axis.
  • This property also requires a color to be set for the shadow.
  • A blur radius can also be set to adjust the depth of the shadow.

Git

  • git clone [paste URL] : clone GitHub repository to local network
  • git remote -v : verify connection between remote & local repositories.
  • git status : check which branch you are in.
  • git checkout -b feature/new-branch-name : create & move to a new working branch.
  • git add -A : add changes to the staging area.
  • git commit -m "add message" : commit changes.
  • git pull origin main : pulls source code to ensure that local has latest commit & changes.
  • git push origin feature/new-branch-name : push committ changes to feature branch.
  • git checkout main : move to main branch.
